About the Data

The Alzheimer's Disease data used in this project was used with permission by The Alzheimer's Disease Prediction of Longitudinal Evolution (TADPOLE) challenge.
The data consists of individuals who have agreed to participate in the Alzheimer's Diease Neuroimaging Initiative (ADNI). The provided features include various biomarkers taken from brain scans as well the results of cognitive tests. Patients are given one of the following diagnoses:
  • CN - cognitively normal
  • MCI - mild cognitive impairment
  • Dementia - probable Alzheimer's Disease

Our Approach

We focused on using the TADPOLE dataset to predict current diagnosis based on the smallest subset of test available for each patient, as well as predicting future onset of Alzheimer's disease for at risk patients. The latter is the more challenging problem that woud allow researchers to identify those individuals who would develop AD later for evaluation of preventive treatments.

Classification Algorithms
We trained and evaluated multiple classification algorithms: Random Forest, Logistic Regression, SVC and Deep Neural Networks. For each classifier, we included different options for pre-processing the data using oversampling and scaling. Deep Neural Networks performed poorely on this dataset, so we dropped those from the model selection.

Feature Selection
Feature Selection was a key step. We focused on collecting rows that had a value in all of the selected features column, and dropped the rest. This resulted in drastically reduced number of rows of data the more features were selected. This highlighted the need for doing feature selection in order to increase the input data size. Features selection is also important in deciding which tests are best predictors of current or future dementia since some of the tests are expensive and/or intrusive.

We used SelectKBest in addition to RandomForest feature improtance list to identify those feature that are best at predicting current and Future dementia diagnosis. Consistently, cognitive tests and PET scan (FDG and AV45) were ranked highest.

Cross Validation
To prevent overfitting by our models, we used K-fold cross validation (k=5) to verify the accuracy reported by our classifiers. The results reported by k-fold verification and our models were very similar.

Predicting Future Diagnosis
The more challenging problem is to predict future diagnosis of AD, which is to identify patients that are at risk for developing AD. To do this, we added a column to each patient's record that was set to the last diagnosis this patient received. Since patients were in the stud for multiple years, in some cases this meant setting a value to AD years before it was diagnosed. Using this approach achived a remarkable accurac of close to 90% for predicting AD, but the model also required a lot of features to make this prediction.

Changes in Values Over Time
We tested the hypothesis that changes over time (e.g. thickness of some regions of the brain as measured on the MRI scans) would be good predictors of AD. In order to test this hypothesis, we added new columns to the data tables that showed % changes over time as well as a rate of change over time. However, these columns did not prove to be better predictors compared to absolute values thickness. This is most likely due to high variation in the thickness results over time.

Challenges Faced

The main TADPOLE spreadsheet consists of over 12,000 rows and 1,900 columns. Many of the columns contained missing data as not all patients underwent the same testing procedures. The abundance of biomarkers made feature selection challenging. In addition, the data was heavily imbalanced resulting in the need to use various oversampling techniques. Data related to MRI and PET scan images showed some unexpected variations, most likely due to different calibrations of the measuring instruments which made it more difficult for the models to rely on changes in MRI measurements over time as a predictor of AD.
